TagUrIt Runtime Package

This folder contains only the files needed to run TagUrIt.

Required runtime files:
- TagUrIt.exe
- leechcore.dll
- vmm.dll
- FTD3XX.dll
- info.db

Optional runtime files:
- symsrv.dll
- dbghelp.dll

How to use:
1. Copy the required DLL files into this folder.
2. Start RainbowSix.exe.
3. Run TagUrIt.exe.
4. If startup fails, check TagUrIt.log in the same folder.

Notes:
- offsets.json.example is a sample file. Rename it to offsets.json if you want to prefill offsets.
- This package intentionally excludes source code and development folders.
